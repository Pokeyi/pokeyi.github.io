# pokeyi.github.io
Simple HTML page for [Pokeyi.dev](https://pokeyi.dev) utilizing GitHub Pages. Also a hub for my other projects.

[![Banner](Pokeyi_Banner.png)](https://pokeyi.dev)

- Icon Set: [Font Awesome](https://fontawesome.com)
